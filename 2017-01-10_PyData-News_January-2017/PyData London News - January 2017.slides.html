POTM - Grumpy¶

  • Google's Python to Go source code transcompiler and runtime
  • Compiles Python source code to Go source code which is then compiled to native code.
  • No bytecode, no VM.
  • Project motivated by a need to improve Youtube concurrent workloads which runs front end on CPython2.7
  • Tradeoffs: no support for C extension modules, but no GIL either
  • https://github.com/google/grumpy

New 3.0 release of PyMC3¶

  • Module for Bayesian Modeling and Probabilistic Machine Learning with Theano
  • In their own words: "This is the first major release of PyMC3"
  • Many upgrades, including:
    • more powerful MCMC samplers
    • Theano for dynamic C compilation and optimizing performance
    • model specification with symbolic formulas (using Patsy)
  • https://github.com/pymc-devs/pymc3/releases/tag/v3.0
